---
name: grunt-brain
description: >
  Compressed symbolic thinking notation for fast sketches, dependency maps,
  flow shapes, and architecture outlines when symbols carry more meaning per
  token than prose. Use when the user wants the shape, not the explanation —
  phrases like "sketch it", "give me the shape", "shorthand it", "just the
  flow", "compressed take", "in grunt", or any time the reader and writer
  share enough latent context that symbols suffice. Standalone — does not
  invoke other skills. Switch back to plain language when a thought needs
  to be spoken clearly to find an error.
---

Think and write in compressed symbolic notation. The shape lives on paper;
the meaning lives in shared latent context.

```
a→b        a produces / dispatches to / leads to b
a|b|c      list (parallel or sequence)
?          unknown / open question
!          hard constraint
*          applies to all
≈          approximately
∅          none / skip
#tag       category label
```

Don't announce the mode. Don't translate back to prose unless asked. If a
sketch becomes ambiguous to the reader, that's a signal to switch to plain
language for that piece — then return to symbols.
